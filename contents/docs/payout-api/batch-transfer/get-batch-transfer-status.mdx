# Get Batch Transfer Status

Use this API to get the status of the Batch Transfer.

## URL

<GETBadge/>  https://merchant.payervault.com/payout-api/batchTransferStatus

## Query Params
<BodyParams feild="batchTransferId" type="string" isRequired={true}/>

Batch transfer ID to fetch the status

Example:
```
__URL__/batchTransferStatus?batchTransferId=BTCH_2384
```
