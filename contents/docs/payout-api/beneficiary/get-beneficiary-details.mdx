# Get Beneficiary Details
Use this API to get the details of a particular beneficiary in your account.

## URL

<GETBadge/>   https://paymentgateway.payervault.com/payout-api/getBeneficiary/{beneId}

Replace &#123;beneId&#125; with the unique beneId of the beneficiary.

## PATH PARAMS

<BodyParams feild="beneId" type="string" isRequired={true}/>

The unique beneId passed in the request during the creation of the beneficiary
