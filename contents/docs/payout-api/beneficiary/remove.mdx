# Remove Beneficiary

Use this API to remove an existing beneficiary from a list of added beneficiaries.

## URL

<POSTBadge/>    https://merchant.payervault.com/payout-api/removeBeneficiary

## Body Params

<BodyParams feild="beneId" type="string" isRequired={true}/>

The unique beneId passed in the request during the creation of the beneficiary
