# Get All Beneficiaries
Use this API to get details of all the Batch Transfers done by you.

## URL

<GETBadge/>  https://paymentgateway.payervault.com/payout-api/allBeneficiary?page={PG_No}

Replace &#123;PG_No&#125; with a valid page number based on the total number of entries.

## Query Params

<BodyParams feild="page" type="number" isRequired={true}/>

You get 10 entries per page. Page Number starts at 1.
